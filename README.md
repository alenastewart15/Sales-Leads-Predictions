# Sales Lead Prediction & Analytics

## Overview
This project analyzes sales lead data using Python, data visualization, and machine learning.

The program:
- Reads lead data from a CSV file
- Categorizes leads as Hot, Warm, or Cold
- Visualizes lead distributions and order values
- Identifies high-priority leads for outreach
- Uses a Random Forest Classifier to predict lead success
- Generates automated sales email examples

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Random Forest Classifier

## Key Results
- Analyzed 200 sales leads
- Identified 78 Hot leads, 62 Warm leads, and 60 Cold leads
- Hot leads averaged over $7,700 in order value
- Built a machine learning model to predict lead success probability
- Created automated outreach email templates
